# Prompts atualizados - Camila, Inbox da Vida Adulta

Use estes prompts em sequência durante a simulação. A ideia é mostrar como uma demanda vaga vira contexto, mini-spec, plano, validação e artefato.

## Prompt 1 - Pedido vago

```text
Minha vida está uma bagunça. Me ajuda a organizar minhas pendências?
```

**Observação para Pedro:** use para mostrar que a IA provavelmente vai responder com conselhos genéricos, uma lista bonita ou uma rotina rígida demais.

## Prompt 2 - Ler os arquivos sem planejar ainda

```text
Atue como uma pessoa que me ajuda a organizar pendências da vida adulta de forma realista, sem criar uma rotina perfeita.

Eu sou a Camila. Anexei arquivos fictícios com meu contexto, comunicações soltas, pendências, contas, assinaturas e janelas de energia.

Antes de criar qualquer plano, leia os arquivos e organize as informações em:

1. fatos confirmados;
2. hipóteses que ainda precisam ser validadas;
3. dúvidas abertas;
4. pendências realmente urgentes;
5. pendências que parecem importantes, mas podem esperar;
6. riscos de montar um plano pesado demais.

Regras:
- não invente informações;
- não tente resolver tudo de uma vez;
- não dê diagnóstico financeiro, médico ou jurídico;
- mantenha um tom humano e prático.
```

## Prompt 3 - Transformar em mini-spec pessoal

```text
Transforme o contexto da Camila em uma mini-spec pessoal.

Inclua:
- contexto;
- problema principal;
- objetivo dos próximos 5 dias;
- restrições;
- princípios da solução;
- escopo;
- fora de escopo;
- critérios de sucesso;
- riscos;
- plano mínimo viável;
- como revisar no final dos 5 dias.

A mini-spec deve reduzir ambiguidade e proteger a Camila de um plano impossível.
```

## Prompt 4 - Classificar pendências

```text
Classifique as pendências da Camila nestas categorias:

- fazer hoje;
- esta semana;
- 15 minutos ou menos;
- agendar;
- adiar sem culpa;
- revisar depois.

Para cada item, explique rapidamente o motivo da categoria, considerando urgência, prazo, esforço, dependências e energia disponível.
```

## Prompt 5 - Criar plano simples de 5 dias

```text
Crie um plano de 5 dias para a Camila.

Regras:
- no máximo 3 tarefas por dia;
- tarefas leves à noite;
- não misturar várias tarefas pesadas no mesmo dia;
- preservar espaço para imprevistos;
- separar tarefas obrigatórias de tarefas adiáveis;
- incluir uma revisão no final.

Não use uma grade rígida de horários. Quero um plano possível, não uma rotina perfeita.
```

## Prompt 6 - Validar contra critérios

```text
Revise o plano usando estes critérios:

1. O plano respeita o limite de 3 tarefas por dia?
2. As tarefas urgentes foram priorizadas?
3. As tarefas pesadas ficaram em períodos de mais energia?
4. Existem tarefas adiadas sem culpa?
5. Há espaço para imprevistos?
6. O plano deixa claro o que não será feito agora?
7. Existe uma revisão final?

Responda em tabela com: critério, status, evidência e ajuste recomendado.
Se algum dia estiver pesado demais, proponha uma versão mais leve.
```

## Prompt 7 - Criar artefato visual

```text
Crie um HTML único para o painel "Inbox da Vida Adulta — Camila".

O HTML deve conter:
- cabeçalho com resumo da situação;
- cards de categorias;
- plano de 5 dias;
- lista de tarefas adiáveis;
- checklist final;
- seção "O que mudou" comparando antes e depois;
- seção "Se o plano sair do controle".

Regras:
- HTML e CSS no mesmo arquivo;
- sem bibliotecas externas;
- visual limpo, moderno e apresentável;
- texto em português;
- usar somente dados fictícios;
- parecer um painel pessoal simples, não uma ferramenta complexa de produtividade.
```

## Prompt 8 - Handoff

```text
Crie um handoff da organização da Camila.

Inclua:
- decisões tomadas;
- plano dos 5 dias;
- tarefas feitas agora;
- tarefas adiadas sem culpa;
- riscos;
- dúvidas abertas;
- próximo checkpoint;
- como ajustar se um dia falhar.

O objetivo é que Camila consiga retomar o plano sem reler toda a conversa.
```
